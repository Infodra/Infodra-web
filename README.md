# Infodra AI Solutions Website

A modern, fully-functional Next.js 14 website for Infodra Technologies showcasing AI-powered business solutions.

**Company**: Infodra Technologies Private Limited  
**Website**: ai.infodratechnologies.com  
**Tech Stack**: Next.js 14 (App Router), TypeScript, Tailwind CSS

## Features

### Pages
- **Home** (/) - Landing page with product showcase
- **Products** (/products) - All products listing
- **Product Detail** (/products/[slug]) - Dynamic product pages with complete information
- **Contact** (/contact) - Contact form and business information

### AI Products
1. **AI Video Translator** - Translate videos into 50+ languages with AI-powered accuracy
2. **AI User Manual Generation** - Auto-generate professional documentation
3. **AI Document Translator** - Professional document translation with 99.5% accuracy
4. **BiZlead Database** - 50M+ verified B2B business contacts

### Components
- Navbar - Sticky navigation header with links and CTA
- Footer - Comprehensive footer with links and company info
- Hero - Hero sections with customizable CTAs
- ProductCard - Product showcase cards
- FeatureGrid - Feature grid layout
- UseCases - Use cases section with multiple columns
- HowItWorks - Step-by-step process display
- PricingPreview - Pricing plans showcase
- FAQ - Accordion-style FAQ section
- CTASection - Call-to-action sections
